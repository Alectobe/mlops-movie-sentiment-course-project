# MLOps Movie Sentiment

Курсовой проект по дисциплине «Операционализация моделей машинного обучения».

## Цель

Построить MLOps-пайплайн для задачи анализа тональности отзывов о фильмах (sentiment analysis) по датасету IMDB.

## Основные компоненты

- Обучение моделей классификации отзывов (positive / negative).
- Отслеживание экспериментов в ClearML.
- Инференс-сервис на FastAPI с эндпоинтами `/health` и `/predict`.
- Упаковка сервиса в Docker-контейнер.

## Структура проекта

- `notebooks/` — эксперименты и ноутбуки (обучение моделей).
- `data/` — данные (сырые и предобработанные).
- `models/` — сохранённые модели и артефакты.
- `configs/` — конфигурационные файлы (YAML и др.).
